## 核心比喻：手机类比（00:18-00:28）

新手别懵！我用手机给你们打个比方，一听就懂：
Node 就是咱们的手机，是运行基础，没手机啥 APP 都用不了；NPM 就是手机里的应用商店，专门用来下载软件。咱们装工具，就跟手机下 APP 一个道理！

---

## 步骤 1：安装 Node（00:28-00:40）

第一步：先装 Node 这个「手机本体」，没装过的朋友，直接打开浏览器搜 Node.js 官网，下载对应版本，全程点下一步，啥都不用改，傻瓜式安装！

---

## 步骤 2：NPM 安装命令 + 避坑（00:40-00:55）

Node 装完，打开命令行工具，输入这条命令：`npm install cloud-code-best -g`，回车等着安装完成！
避坑！要是安装失败、外网打不开，要么换国内 NPM 源，要么开魔法上网，俩方法都能解决，亲测有效！

---

## 步骤 3：启动项目（00:55-01:05）

安装完啥也别管，直接输 `ccb` 回车，瞬间进入工具界面，到这步就说明安装成功了，是不是超简单！
