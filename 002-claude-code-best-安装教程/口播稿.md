# Cloud Code Best 安装教程 — 口播稿（优化版）

> 时长：约 1 分 40 秒
> 风格：个人经历 + 实操，口语化，有梗
> 钩子：邪修 + 封孙子

---

## 钩子（00:00-00:08）

我感觉我快成 Claude Code 邪修了——封我跟封孙子似的，5 个号全没，5000 块打水漂。后来我干了一件事，它再也封不了我了。今天教你，一分钱不花。

---

## 版本选择（00:08-00:17）

先说清楚，Cloud Code Best 不是官方原版，是泄露代码的优化版本。安装分两条路：想研究源码的选源码版，90% 的人只想赶紧用上——直接走 Node 快速安装版。今天只讲这个。

---

## 手机比喻（00:17-00:27）

先别懵。我打个比方你就明白了——Node 就是你手机，没手机什么都跑不了。NPM 就是应用商店，下软件用的。装这个工具跟下个 APP 一个道理。就这一句话，后面你全懂。

---

## 步骤 1：装 Node（00:27-00:39）

第一步，装 Node——就是先搞一台「手机」。浏览器搜 Node.js 官网，下载、安装，全程下一步，不用改任何设置。

---

## 步骤 2：安装命令（00:39-00:52）

Node 装好之后，打开命令行，敲这一行：

`npm install cloud-code-best -g`

回车，等着。装不上？两个办法：换国内 NPM 源，或者开魔法上网。两个我都试过，都行。

---

## 步骤 3：启动（00:52-01:02）

装完直接敲 `ccb`，回车。看到主界面就说明成了。

---

## 步骤 4：配置密钥（01:02-01:17）

进去之后敲 `login`，绑 DeepSeek 的密钥。为什么选 DeepSeek？10 块钱够你用一个月，自己试一下就知道了。配好就能用，测过了。

---

## 收尾（01:17-01:27）

源码版适合想钻进去研究的，小白不用碰。装不上的评论区打 `ccb`，我帮你看。有用的话点个关注，后面还有更多不花钱的工具。
