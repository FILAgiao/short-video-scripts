## 实拍实操指导

### 1. 拍摄设备

- 本人出镜：手机竖屏，后置更清晰，固定机位
- 电脑录屏：Windows 用 Xbox 录屏（Win+G），Mac 用 QuickTime，命令行字体 14-16 号

### 2. 语速 & 语气

- 钩子部分：带自嘲，像跟兄弟吐槽，不用太正经
- 比喻和命令：放慢，咬字清楚
- 整体：比你平时说话稍快一点，但关键地方要停顿
- 别念稿——记住每段的要点，用自己话说

### 3. 电脑录屏要点

- 提前清理桌面和浏览器标签
- 命令行字体调大（16 号）
- 鼠标移动放慢
- 素材先录好，后期对着画面配音更灵活

### 4. 后期剪辑

- 转场：钩子用「闪白」，其余硬切
- BGM：轻科技感背景音，人声突出，音量 -20dB 以下
- 全程字幕，命令和关键词高亮

### 5. 避坑清单

- ❌ 不提「破解」「盗版」「白嫖」
- ❌ 不露出个人 API Key
- ❌ 不引导非官方付费渠道
- ✅ 只说「非官方原生」「泄露代码优化版」「免费平替」
