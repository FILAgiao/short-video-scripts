## 实拍实操指导

### 1. 拍摄设备

- 本人出镜：手机竖屏拍摄，后置摄像头更清晰，桌面架好手机，固定机位别晃动
- 电脑录屏：Windows 用 Xbox 录屏（Win+G），Mac 用 QuickTime，录屏时把命令行字体调大（14-16 号），方便手机观看

### 2. 语速 & 语气

- 整体语速：比平时说话快一点，符合抖音节奏，但关键步骤（命令、比喻）放缓
- 语气：像跟朋友分享干货，别念稿，自然有停顿
- 关键命令（npm install / ccb / login）：放慢语速，一个字一个字说清楚

### 3. 电脑录屏要点

- 提前清理桌面和浏览器标签页，只留需要展示的窗口
- 命令行字体调大（推荐 16 号），手机端才看得清
- 鼠标移动放慢，给观众跟上的时间
- 所有录屏素材先录好，再对着素材配音，后期剪辑更灵活

### 4. 后期剪辑

- 转场：钩子部分用「闪白」，其余用「硬切」，不要花哨
- BGM：轻量科技感背景音，人声突出，音量控制在 -20dB 以下
- 全程加字幕，命令和关键词高亮标注

### 5. 避坑清单

- ❌ 不要提「破解」「盗版」「白嫖」等词 → 容易限流
- ❌ 不要在视频里露出任何个人 API Key → 截图打码
- ❌ 不要引导用户去非官方渠道下载付费破解版
- ✅ 只说「非官方原生、泄露代码优化版」「免费平替」
