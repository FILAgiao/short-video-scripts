# 短视频脚本库

个人短视频创作脚本存档。每个视频一个文件夹，内含口播稿和画面稿。

## 目录结构

```
├── 001-claude-code-免费用/
│   └── 口播稿.md              # 纯文案，对镜直接录
│
├── 002-claude-code-best-安装教程/
│   ├── 口播稿.md              # 纯口播，方便提词
│   └── 画面稿.md              # 逐秒分镜，含拍摄指导+字幕贴纸
│
└── README.md
```

## 文件说明

- **口播稿.md** — 纯口播文案，适合提词器/快速录制
- **画面稿.md** — 完整实拍全案，含口播+画面分镜+字幕贴纸+拍摄指导+后期要点，照着拍出片
